# ustc-wlt
A non-interactive POSIX shell script for USTC wlt network portal login

## Install

[RPM packages](https://software.opensuse.org//download.html?project=home%3ASam0523&package=ustc-wlt)

## Depends

* /bin/sh
* coreutils
* sed
* grep
* w3m

## Configuration file location

* /etc/wlt.conf
* ~/.config/wlt.conf
